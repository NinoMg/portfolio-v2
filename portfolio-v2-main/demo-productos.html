Demo - Sistema de Gestión de Productos

Aplicación desarrollada en PHP y MySQL ejecutada en entorno local.

Este sistema permite gestionar productos mediante operaciones CRUD completas, incluyendo creación, edición, eliminación y visualización de datos.

🔧 Tecnologías: PHP, MySQL, Bootstrap
⚙️ Ejecutado en entorno local con Apache (XAMPP)

Pantalla de inicio de sesión para acceso al sistema.

Panel principal con listado de productos almacenados en base de datos.

Formulario de edición con datos precargados y actualización dinámica.
